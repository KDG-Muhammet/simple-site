Dit tooltje doet een validatie van alle htm(l) en css bestanden in de huidige directory en alle subdirectories volgens de html5 standaard.
Je pakt het zip bestand best uit in de de directory waarin de git-directory staat.
Zo zullen alle bestanden in het groeiproject en examenproject gevalideerd worden.

Het resultaat komt in een tekst bestand.
